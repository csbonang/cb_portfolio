Improving Accessibility Options on Genshin Impact iOS

Description
 This report summarizes a study investigating Genshin Impact's accessibility features and their effectiveness as assessed by a broad range of players. The study employed surveys, interviews and usability testing. The usability testing was conducted on a single platform (mobile devices running the iOS operating system) and used a well-defined subset of the game called Commission Quests. The report provides both qualitative and quantitative analyses of the collected data.The report also examines potential accessibility enhancements such as, color blind mode, improvements to the user interface, and text scalability
